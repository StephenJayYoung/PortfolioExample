Why Bootstrap?


1. Responsiveness (the grid system/class system)

2. Less CSS

3. Ability to make things quicker

4. Good documentation

Steps For Downloading a Theme


1. https://pages.github.com/

2. Choose a theme, download the zip file, and put it in your local folder

3. Make sure the index.html file is visible (or else gh pages won't work!)

4. Pull up your text editor, open your site in your browser, and get to work!

Tips For Manipulating a Theme


1. Read the Bootstrap Documentation (understanding the class and grid system is very helpful)

2. Keep it simple

3. Use Chrome Developer Tools

4. Peruse some other bootstrap themes, and if you like something you see on one of those sites, copy that code into your text editor and use that

4. Keep your code clean and organized

4.Git Commit!
